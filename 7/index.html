Позиционирование в верстке

Основные концепции и практики

Сегодня мы поговорим о позиционировании в CSS — ключевом инструменте для создания сложных макетов.

Типы позиционирования

  • static
  • relative
  • absolute
  • fixed
  • sticky
CSS предлагает 5 типов позиционирования: static, relative, absolute, fixed и sticky. Каждый из них имеет свои особенности и области применения.

Static

Без изменений
Static — это значение по умолчанию. Элементы располагаются в обычном потоке документа без каких-либо изменений.

Relative

Смещение относительно себя
Relative позволяет смещать элемент относительно его обычного положения. Остальные элементы остаются на своих местах.

Absolute

Относительно ближайшего позиционированного родителя
Absolute позиционирует элемент относительно ближайшего родителя с position: relative, absolute или fixed. Если такого нет, то относительно viewport.

Fixed

Фиксированное положение относительно viewport
Fixed позиционирует элемент относительно окна просмотра. Элемент остается на месте даже при прокрутке страницы.

Sticky

Переключение между relative и fixed
Sticky комбинирует поведение relative и fixed. Элемент начинает вести себя как fixed, когда достигает определенной точки при прокрутке.

Пример кода


.example {
  position: relative;
  top: 20px;
  left: 30px;
}
Вот пример использования relative позиционирования. Элемент смещается на 20px вниз и на 30px вправо от своего обычного положения.

Заключение

Позиционирование — мощный инструмент для создания гибких макетов.

Правильное использование позиционирования помогает создавать адаптивные и удобные интерфейсы. Важно понимать, как работают разные типы позиционирования.
